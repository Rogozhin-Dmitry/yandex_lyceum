Мои проекты

В этом разделе я немного напишу про свои проекты.
Я участвовал в проекте "Коворкинг Байтика". Этот проект подразумевал, что обычные школьники выполняют полноценные рабочие задачи для сторонних клиентов (например, Администрация Троицка). Вот несколько подобных проектов, к сожалению, на гит я их выложить не могу, поэтому просто покажу их тут.

Первое, что хотелось бы показать, - достаточно интересный проект, это main файл бота в телеграм, который собирает у людей обращения по поводу борщевика в Новой Москве. Для этого пользователю необходимо или передать своё местоположение, или указать его на специальном сайте. После того как обращение будет сформировано, оно переконвертируестя в doc файл, и отправится на почту заказчика. Сам проект реализован почти на фласк, карта же собой представляет элемент js api Яндекс карт, к которому просто привязан простой обработчик на js. Здесь используется callback api telegram.

Второе - это тот же код, но написаный на longpool api с использованием библиотеки aiogram. Во время того заказа пришлось всё переделывать раз 10, клиент сам не понимал, что хочет, в результате получились 2 таких кода.

Третье - это вот такой вот бот, он писался для человека, абсолютно не сведующего в програмировании. Он имеет оооочень гибкие настройки через конфиг файл. Суть этого бота в конечном счёте сводится к выбору пользователем картинки, и нанесению на неё пользовательской надписи (эдакий мини paint). Данный бот реализован на longpool api от vk, с помощью библиотеки vk_api, для нанесения надписи ипользуется PIL.

Ещё хотелось бы показать такую простенькую библиотеку, упрощающую работу с сервисом vk_coin. Для него была хорошая библиотека на python, однако по каким-то соображениям она была удалена из pip. Поэтому для облегчения написания одного из своих проектов, я её восоздал, это было не сложно, так как все методы имеют чёткое описание.

Ещё есть вот такой бот для телеграм. Этот бот собирает обращения у обычных граждан и отправляет их администрации города Троицка. Благодаря большому количеству клавиш, можно отправить обращение практически в любую организацию, по буквально любому поводу. Этот бот написан на longpool api telegram с использованием библиотеки aiogram.

Также конец хотелось бы продемонстрироваь мою первую игру. Я её написал во второй половине первого года Яндекс лицея, к сожалению, у меня в тот момент не было особого понимания ни принципов ООП, ни о разработки игр, однако у меня было колоссальное количество времени и огромный интузиазм. В результате у меня получилась игра-песочница с видом сверху. Она сделана на pygame, с использованием PIL.
